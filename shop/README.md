1) Реализована реактивная форма для заполнения данных о пользователе, для подальшей отправки продуктов
2) Добавлена нужная валидация для всех полей пользователя, в том числе - проверка электронной почты на нужный паттерн, проверка номеров телефонов на нужный паттерн
3) Самые нужные поля установлены как Required, так же, для полей firstName, lastName, email установлено минимальное количество символов - 3
4) Адрес необходимо заполнять только, если отмечен checkbox 'I would like to recieve products by Delivery Service'
5) Реализована возможность динамического добавления Form Control элементов, для заполнения номеров телефонов пользователя
