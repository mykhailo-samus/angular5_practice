1) Реализованы все CRUD-операции по работе с продуктами на основании Promise и Observable подходов
2) Для получения списка продуктов и его деталей, используется productPromiseService, для изменения/добавления продутка - productObservableService
3) Реализован декоратор для автоматической отписки от событий
4) Реализован интерсептор, который считает время запросов по адресу 'http://localhost:3000/products'

Для запуска локального backend-сервера, необходимо в папке shop\src\app\db выполнить команду:
json-server --watch db.json
